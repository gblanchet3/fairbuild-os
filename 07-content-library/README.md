# Content Library

All content FairBuild puts into the world — organized so we can (a) not repeat ourselves, and (b) learn what works.

## Structure

- `linkedin-posts/` — organic LinkedIn content, Kendall-first-person
- `ad-copy/` — paid ad copy, organized by campaign and platform

## Naming convention

`YYYY-MM-DD-short-slug.md` — e.g. `2026-04-28-180k-indemnification-story.md`

## File template

```markdown
# [Title / hook]

- **Channel:** (LinkedIn organic / LinkedIn ad / Instagram ad)
- **Campaign:** (if ad)
- **Date posted:**
- **Status:** (draft / scheduled / live / archived)
- **Performance:** (update after 7 days: impressions, clicks, CTR, CPL)

## Body

[content]

## Notes

- Source story: (link to 06-sales-playbook or customer file)
- Variant of: (if A/B test)
- What we'd change: (after data comes in)
```
